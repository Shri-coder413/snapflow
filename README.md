## Snapflow

Snapflow is a simple **short-video feed web app** built using **HTML, CSS, and JavaScript**.
It demonstrates how modern reel-style feeds work on the web.

---

### What this project does

* Shows videos in a **vertical scrolling feed**
* Automatically **plays a video when it comes into view**
* Pauses the video when it goes out of view
* Allows basic **like and follow interactions** (UI only)

---

### Why this project exists

This project was created to practice and demonstrate:

* Core **JavaScript DOM manipulation**
* Event handling and user interactions
* Performance-friendly video playback using **Intersection Observer**

It is meant for **learning and portfolio use**, not as a full social media application.

---

### Tech Stack

* HTML
* CSS
* Vanilla JavaScript
